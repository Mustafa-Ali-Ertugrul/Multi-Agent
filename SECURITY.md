# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly.

- **Do NOT** open a public GitHub issue.
- Email the maintainer at: `ertugrulmustafaali@gmail.com`
- Include a clear description of the issue and steps to reproduce.

You will receive a response within 72 hours. Please do not disclose the
vulnerability publicly until a fix has been released.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | ✅        |
| < 0.2   | ❌        |
